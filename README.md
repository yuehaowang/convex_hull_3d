# Convex Hull 3D

My implementation of Incremental Convex Hull algorithm for 3D objects.

## Dependencies

- Python 3.7
- Open3D
- NumPy
- tqdm

## Usage

```
optional arguments:
  -h, --help            show this help message and exit
  --file FILE           The target model file.
  --vis                 Whether visualize the result?
  --save_path SAVE_PATH
                        The saving path of the result.
```
